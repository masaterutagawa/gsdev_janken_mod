# じゃんけん課題の改造用

自分の提案価値を考えてみるためのアプリ

## DEMO

- https://masaterutagawa.github.io/gsdev_janken_mod/

## 紹介と使い方

- 最初に自分の名前を入力し、あとは質問に対して回答していくだけ

## 工夫した点

- Jquey でフォームの値を取得して表示させていくところ
- 自分の乏しい知識でどれだけ実装できるか？
- コピペではなく、自分なりに考えてコードをかく

## 苦戦した点

- 質問の組み合わせがうまくできないので、課題提出後にここは更に考えます
